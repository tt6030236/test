Refgenie Data Channel

This site can be used as a Refgenie data channel -- source of asset classes and recipes.

1. Connect

Add this data channel by executing the following command:

refgenie data_channel add  https index.yaml

2. Sync

Add all asset classes and recipes published by this data channel by executing the following command:

refgenie data_channel sync

To learn more about the asset classes and recipes offered by this data channel, view the index.yaml file.

About Refgenie

Refgenie is a reference genome asset manager that makes it easy to organize, retrieve, and build genome analysis resources. It provides a standardized interface for accessing common bioinformatics reference assets across environments.

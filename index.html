This site can be used as a Refgenie data channel for managing genomic assets.

Connect to this data channel

Add this data channel by executing the following command:

refgenie data_channel add channel_name https index.yaml

About Refgenie

Refgenie is a reference genome asset manager that makes it easy to organize, retrieve, and build genome analysis resources. It provides a standardized interface for accessing common bioinformatics reference assets across environments.
